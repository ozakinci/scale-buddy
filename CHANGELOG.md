# Changelog

All notable changes to ScaleBuddy are documented in this file.

## [2.1.1] - 2026-04-01

### Fixed
- **HTML Entity Encoding Bug (#1)** - Fixed corrupted placeholder text in result cards showing `â€"` (UTF-8 encoding error for em-dash)
- **Neapolitan (N6) Chord Definition (#2)** - Corrected chord intervals from `[0, 4, 7]` (Major triad) to `[0, 3, 6, 10]` (actual Minor 7th chord in classical theory)
- **Neapolitan Minor Scale Metadata** - Corrected `SCALE_DEGREE_CHORDS` to match actual interval structure (now matches `harmonicMinor` correctly)
- **Neapolitan Major Scale Metadata** - Corrected `SCALE_DEGREE_CHORDS` to match actual Major scale intervals

### Documentation
- Added clarifying comments in code for duplicate scales:
  - Chinese scale (regional variant, identical to Mongolian scale)
  - Mongolian scale (regional variant, identical to Chinese scale)
- Documented that Chinese and Mongolian represent the same scale with different cultural naming conventions

### Known Issues - Non-Breaking
- Some world scales (Balinese, Japanese, Hirajoshi, Iwato, Yo) have identical `SCALE_DEGREE_CHORDS` entries - these are intentional as different scales can produce equivalent chord patterns at different scale degrees. The W/H patterns remain accurate for each unique scale.
- Edge case: Double flats/sharps (e.g., B#, Cb) not currently generated by enharmonic spelling. Not relevant to current scale set.
